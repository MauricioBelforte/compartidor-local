# Sistema de Documentación del Proyecto

Este proyecto usa un sistema de documentación estructurado para rastrear el estado actual y el historial de componentes.

## Estructura

- **Raíz (`DOCUMENTACION/`):** Documentación general vigente del proyecto
- **Plan Inicial:** Documentación original del proyecto (referencia histórica, no modificar)
- **Componentes numerados:** Cada componente tiene su carpeta con `plan-inicial/` y `plan-actual/`
